Anime.js

The ultimate way to animate a website.

Anime.js is a JavaScript plugin that helps build flowing animations for web projects. Anime.js tries to make animations simpler while giving extensive control over how things move. It has been used in interactive user interfaces, lively visual infographics plus inventive UI web work. Their website offers many tutorials on how to animate.
